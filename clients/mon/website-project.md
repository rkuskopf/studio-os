## Page Breakdown

### Home
- Nav / header (universal)
- Single video container — montage of work (above fold)
- Footer (universal on every page)

### Work
- **Desktop:** Clicking "Work" goes to first project. Single column image grid. Left-hand submenu to navigate between projects (max 8). Nothing fixed — nav, submenu, description all scroll with page.
- **Mobile:** Submenu becomes dropdown when clicking "Work"
- Projects managed via Framer CMS (she can add/edit herself)
