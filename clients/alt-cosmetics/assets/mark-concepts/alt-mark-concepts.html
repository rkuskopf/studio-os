Identity Exploration — Mark Concepts

ALT.

The key insight

The Loewe logo works because "L" is asymmetric — when you reflect an asymmetric letter 4 ways, the strokes cross at the centre and create an interlocking pattern. Reflecting a symmetric letter (like "A") just makes a boring hourglass. ALT. already has an L in it. The anagram mark direction should use "L" as its base letterform — it will look exactly like the Loewe construction in principle, but it's yours. Directions below show both the L-anagram and the other two directions (cartouche, ferronnerie) as concept sketches.

Monday Plan — Fastest Path to Mockups

Tonight / Saturday AM — Get the marks

Anagram (Dir 01): Open this HTML in Illustrator via File → Place. The SVG arm path is your starting point. Trace it with the Pen tool using a calligraphic brush, then use Object → Transform → Reflect. 30–40 min.

Cartouche (Dir 02): Don't generate it — buy it. Go to creativemarket.com and search baroque cartouche vector or ornate oval frame vector. $15–25, you get a perfect Illustrator file. Drop your A letterform in. Done in 10 minutes.

Ferronnerie (Dir 03): Same approach — search art nouveau frame vector or wrought iron ornament vector on Creative Market or envato.com.

Saturday PM — Get the mockups

Chrome tube mockup: Go to unblast.com (free) or yellowimages.com and download a cosmetic tube mockup PSD. Search cosmetic tube mockup PSD. Open in Photoshop, double-click the smart object layer, paste your mark. Save. Done in 5 minutes per direction.

Alternative (even faster): Go to smartmockups.com — upload a PNG of your mark, it places it on a 3D tube render in the browser. No Photoshop needed. Free tier available.

Result: 3 directions × chrome tube = 3 mockup images ready for Anh.

Note on the A vs L question

The anagram direction doesn't have to use the literal letter A. The Loewe logo uses L — but it's known as the Loewe anagram, not just the Loewe L. The mark represents the brand, not the initial. Using L from ALT. gives you the same interlocking dynamic that makes the Loewe construction work. If you want A specifically, use just one diagonal stroke of the A (not the full letter) — that asymmetric stroke will reflect in an interesting way. The full symmetric A reflects into an hourglass. One diagonal arm of A reflects into something much more alive.
