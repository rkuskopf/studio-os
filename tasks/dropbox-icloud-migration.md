# Data / Folder Structure — Dropbox to iCloud Migration

## Problem
Need to move all files from Dropbox to iCloud before Dropbox deletes the data (~50 days from when this was written).

**Blockers:**
- Not enough storage on local hard drive to download files from Dropbox first
- Every transfer method tried so far hasn't worked
- Dropbox support ignored the message about being locked out of data

## Context
Dropbox is effectively holding the data hostage — can't download without local storage, can't transfer directly to iCloud.

## Goal
Get all Dropbox files safely into iCloud without needing to store them locally first.

## Options to Explore
- Dropbox → iCloud direct transfer via a third-party service (e.g. MultCloud, Cloudsfer)
- Temporarily expand Dropbox storage to buy time
- Use an external hard drive as intermediary
- Escalate with Dropbox support (reference data hostage / consumer rights angle)
