# Weekly Review

Run my end-of-week review for KSPF.

1. Read `TASKS.md`, `CLAUDE.md`, and active project files
2. Summarise what was completed this week vs what was planned
3. Flag anything that's slipped, blocked, or needs a decision
4. Ask me: anything that happened this week not captured in the files?
5. Prompt me to move completed tasks to the **Done** section of `TASKS.md`
6. Ask if I want to archive old Done items to keep the file clean
7. Produce a brief "week in review" summary — useful for my own records or sharing

End by asking: what's the one thing I want to make sure happens next week?
