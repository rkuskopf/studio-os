# New Project

Set up a new client or internal project in this repo.

1. Ask me for: project name, client name (if applicable), project type (branding / web / strategy / other), key deadline, and a one-line brief
2. Create a new markdown file in the repo root named `[project-slug].md` using the structure from `alt-cosmetics.md` as a template
3. Add the project to the projects table in `CLAUDE.md` with status "Active"
4. Add an initial task to `TASKS.md` under **Up Next**: "Kick off [project name]"
5. Confirm all changes and show me what was created

Keep filenames lowercase with hyphens. No spaces.
