# studio-os

**This is the operating system for KSPF** — Rowan Kuskopf's creative studio (kspf.au). It is a plain-text knowledge base that holds studio context, client information, active project files, and task lists.

It manages three distinct life/business streams:
- **Stream 1: KSPF** — design business (client work, lead gen, studio ops)
- **Stream 2: Slowtides** — music business (production, practice, releases)
- **Stream 3: Personal** — wellness, health, personal development

It is designed to be read by AI assistants. When you are given access to this repository, **read all the files listed below before responding**. They are short markdown files and together give you the full picture of who Rowan is, what the studio does, what is currently happening, and who the clients are.

---

## How to use this repo with AI

1. **Paste the repo URL** (or upload/attach the files) into your AI session.
2. Ask the AI to "read all files in this repo" or "read the context files before we start."
3. The AI should start with `README.md`, then read the files in the order listed below.
4. Once read, the AI has full context and can help with tasks, writing, planning, and decisions without needing re-explanation every session.

> **No connector is needed.** These are plain `.md` (Markdown) text files — every AI assistant (ChatGPT, Claude, Gemini, etc.) can read them directly when you paste their contents or attach them to a conversation. If an AI says it "can't read the files," paste the content of the relevant file(s) directly into the chat.
