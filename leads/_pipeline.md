# Lead Pipeline

**Source of truth:** This file + Asana "Lead Generation" project are kept in sync.
- **This file:** Quick-view reference, notes, context, history.
- **Asana:** Where actions live — due dates, task assignments, follow-up reminders.

---

## Stage Definitions

| Stage | Meaning | Asana Section |
|-------|---------|---------------|
| **Identified** | Name found, not yet researched | Identified |
| **Researched** | Know their work, found contact, ready to reach out | Researched |
| **Outreach Sent** | First message sent, no reply yet | Outreach Sent |
| **Replied** | They responded | Replied |
| **In Conversation** | Active dialogue, scoping potential | In Conversation |
| **Proposal** | Brief/quote sent | Proposal |
| **Won** | Engaged, project starting | Won |
| **Lost / Paused** | Not proceeding (note why) | Lost / Paused |

---

## Asana Sync Guide

Each lead = one Asana task in the **Lead Generation** project.

**Task name format:** `[Stage] Lead Name — Company`
Example: `[Outreach Sent] Jordan Lee — Maison Dey`

**Task description should include:**
- Contact details
- Why they're a fit
- Links (Instagram, site)
- Outreach notes + dates

**Move tasks between sections** in Asana to match stage changes here.

**Weekly habit:** During Sprint Planning (Mon 8am), scan both this file and Asana — update stages, log last contact dates, check what needs a follow-up.
