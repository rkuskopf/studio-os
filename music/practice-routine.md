# Slowtides — Practice Routine

A mandatory structure for music practice and production. Treats music as a craft that requires consistent, deliberate practice — not just inspiration-based sessions.

---

## Core Principles

1. **Show up daily.** Even 20 minutes beats skipping.
2. **Separate practice from production.** Practice = skill building. Production = making tracks.
3. **Make it non-negotiable.** It's in the calendar: 4:00–5:00pm daily.
4. **Track what you work on.** Log the session. Patterns emerge.

---

## Daily Structure (1 hour block, 4:00–5:00pm)

Not every element every day. Rotate through the four zones across the week.

| Zone | Time | What it is |
|------|------|------------|
| **Instrument practice** | 20 min | Keys technique, chords, scales, melodic improvisation |
| **Music theory** | 10 min | Study one concept, apply to piano or DAW |
| **DAW / sound design** | 20 min | Experiment in Logic — no goal, just exploration |
| **Listening** | 10 min | Active listening to reference track (headphones, full attention) |

## Instrument Practice (Keys)

### Warm-up (5 min)
- Scales: C major → relative minor → one modal scale
- Chord inversions: I–IV–V–I across all inversions in one key

### Main Practice (15 min — rotate weekly)
**Week A:** Left-hand independence — hold a pad/drone with left, melody with right
**Week B:** Chord voicings — open voicings, drop-2, colour tones (9ths, 11ths)
**Week C:** Slow melodic improvisation — stay in one key, play slowly and intentionally
**Week D:** Play along to a reference track — match the feel, not note-for-note

---

## DAW / Sound Design Practice

These are **practice exercises**, not production sessions. Cap them at 20–30 min.

### Rotation (do one per session)
- **Granular exploration** — open Alchemy, granular mode. Design one evolving pad from a simple source.
- **Reverb space design** — build one custom reverb chain. What room does this track live in?
- **Drum texture** — make one drum loop that doesn't sound like a loop. Add variation, ghost notes, decay.
- **Layer study** — take a reference track section, reconstruct the layers (bass, pad, melody, texture). Don't copy — understand.
- **One-knob exercise** — open a synth, start with a basic sine/saw, change only one parameter at a time for 20 min. Listen.

## Music Theory Curriculum (self-paced)

Work through these concepts in order. One per week, applied practically:

1. ✅ Chord progressions in major/minor (you know this)
2. [ ] Suspended chords (sus2, sus4) and when to use them
3. [ ] Borrowed chords (modal interchange) — borrowing from parallel minor
4. [ ] Secondary dominants — creating stronger movement
5. [ ] Modes in practice — Dorian, Lydian, Mixolydian (ambient uses Lydian heavily)
6. [ ] Harmonic rhythm — how often chords change affects emotional weight
7. [ ] Counterpoint basics — two independent melodic lines
8. [ ] Polyrhythm — 3 against 4, 5 against 4 (relevant for rhythmic ambient)

**Resource:** Use Piano With Jonny (YouTube), musictheory.net, or 12tone (YouTube) — all free.

## Active Listening Practice

Not passive background music. Full-attention listening to reference tracks.

### How to do it
1. Pick one track from the [creative inputs list](creative-inputs.md)
2. Headphones, eyes closed, full track
3. After: note 3 things you heard — a texture, a transition, a sound design choice
4. Optional: open DAW and try to recreate one of those elements

### Current Listening Queue
- Nils Frahm — *Says* (space and dynamics)
- Jon Hopkins — *Emerald Rush* (rhythmic texture, build/release)
- Rival Consoles — *Untravel* (sound design approach)
- Max Richter — *On the Nature of Daylight* (emotional arc, restraint)

## Production Session vs Practice Session

Keep these **separate** — don't let one eat the other.

| | Practice Session | Production Session |
|-|-----------------|-------------------|
| **Goal** | Build a skill | Advance a track |
| **Duration** | 20–60 min | 2–4 hours (Deep Focus block) |
| **Output** | Learning | Track progress |
| **Mindset** | Experimental, okay to fail | Intentional, decision-making |
| **When** | 4:00–5:00pm daily | 2:00–4:00pm (deep focus) |

Production sessions for Midnight and other active tracks happen in the **2:00–4:00pm Deep Focus block**, not in the practice hour.
