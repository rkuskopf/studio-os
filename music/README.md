## Blockers

### Can one Asana workspace link to two GitHub repos?

**Finding:** Yes — Asana's GitHub integration is at the **project** level, not workspace level. You can connect different Asana projects to different GitHub repos. This means:
- KSPF Asana project → `rkuskopf/studio-os` (design business)
- Slowtides Asana project → `rkuskopf/north-star` (music)

**Action:** When setting up the `north-star` repo, create a dedicated Asana project for Slowtides and link it independently.
