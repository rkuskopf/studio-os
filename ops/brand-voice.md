## Hard Constraints for AI

When using Claude or any AI tool with this repo as context, these rules must hold:

1. **Never promise timelines not agreed in writing** — if a date hasn't been confirmed, don't state one
2. **Match sentence length to the medium** — short for outreach emails, longer for proposals and case studies
3. **No hollow openers** — don't start with "I hope this finds you well" or similar
4. **Always anchor to value, not hours** — talk about outcomes, not effort
5. **Match the client's vocabulary** — if they said "brand identity", use that, not "visual identity system"
